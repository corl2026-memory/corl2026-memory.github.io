# Memory for Robot Foundation Models — CoRL 2026 Workshop

Website for the **Memory for Robot Foundation Models** half-day workshop at the
Conference on Robot Learning (CoRL) 2026 — Austin, TX, USA · November 9, 2026.

## Deploy

This is a single static page (`index.html`) with no build step — Tailwind, fonts,
and avatars load from CDNs. To publish on GitHub Pages, push this repo to a
`*.github.io` repository (or enable Pages on any repo) and serve from the root.

Local preview: just open `index.html` in a browser.

## Editing content

All content lives in `index.html`. The most common edits:

- **Speakers** — the `#speakers` section. Replace the "To be announced" placeholder
  cards with confirmed speakers; swap the `ui-avatars.com` URL for a real photo.
- **Important dates** — the dates card inside the `#cfp` section (currently `TBA`).
- **OpenReview link** — the "Submit on OpenReview" button in `#cfp` (currently `#`).
- **Schedule** — the timeline `<ol>` in the `#schedule` section.
